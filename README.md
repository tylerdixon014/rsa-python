# rsa-python

## Description
Implementing a basic RSA encryption method in Python. 

## Features
 - Prime generation
 - Public / private key generation
 - Encryption / decryption of a message
 - Modular arithmetic functions (computing d)

## Getting Started
Clone repo and run 'rsa.py'.
